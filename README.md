# Site Institucional – Duas Barras - Granitos

Website desenvolvido com foco na apresentação institucional de uma empresa do segmento de mármores e granitos, destacando serviços, produtos e portfólio de forma clara, moderna e acessível.

## Funcionalidades

* Apresentação de serviços e materiais
* Exibição de portfólio de projetos
* Layout responsivo
* Navegação simples e intuitiva

## Tecnologias

* HTML
* CSS
* Bootstrap

## Objetivo

Proporcionar uma presença digital profissional, facilitando a divulgação dos serviços e fortalecendo a comunicação com clientes.
